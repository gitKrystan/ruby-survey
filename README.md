# Survey

#### By Krystan Menne and Patrick Sullivan

## Description
Ruby web app that allows users to create and take surveys.

## Technologies Used
HTML, CSS, Ruby, ActiveRecord, PostgreSQL, rspec, capybara, sinatra.

## Installation
Clone repository.  
Run Postgres.  
In terminal:

```
bundle install  
rake db:create
rake db:migrate  
rake db:test:prepare
ruby app.rb
```
